# Neural Network Modeling & K-Means Clustering

This project includes:
- Deep Neural Network (TensorFlow/Keras) for modeling diode V-I relationship
- Custom implementation of K-Means clustering from scratch
- Visualization of predictions vs real data

## Key Features
- Physics-based data generation
- Early stopping & learning rate scheduling
- Convergence detection in clustering

## Tech Stack
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
